2) Поиск по маске это урезанный поиск по регулярному выражению, где "*" - любое количество символов, а "?" - один символ.
Если в качестве маски код принимает "*.jav?", то нужно "*" и "?" заменить на эквивалентные регулярные выражения. Скомпилировать их и искать по регулярке.

3) Когда ищите по регулярке, надо использовать Pattern.compile() и Matcher, потому, что matches() каждый раз компилирует шаблон, что сказывается на производительности.
Метод find() ищет вхождения выражений в тексте, переданном методу Pattern.matcher(text) при создании Matcher
Источник: https://java-blog.ru/osnovy/klass-java-matcher

4) Формирование предиката и поиск стоит вынести в отдельный метод, который будет возвращать результат. Запись в файл также в отдельный метод. Это принцип разделения ответственностей.
